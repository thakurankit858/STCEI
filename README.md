**STCEI Website Clone ‚Äì HTML, CSS, Bootstrap & Animation**

This project is a fully responsive clone of the **STCEI (Smart Tech Computer Education Institute) official website**, created using HTML, CSS, Bootstrap, javaScript and animation libraries.
It recreates the layout, design, and visual experience of the original website with modern UI and smooth animations.

---

**Features**

* Fully responsive for mobile, tablet, and desktop
* Clean and modern UI
* Built using Bootstrap 5
* Smooth scroll and fade animations
* Reusable components (Navbar, Footer, Cards, Sections)
* SEO-friendly HTML structure
* Cross-browser compatibility

---

**Technologies Used**

| Technology            | Purpose                      |
| --------------------- | ---------------------------- |
| **HTML5**             | Page structure               |
| **CSS3**              | Custom styling               |
| **Bootstrap 5**       | Layout & UI components       |
| **Animation Library** | Smooth transitions & effects |
| **Font Awesome**      | Icons                        |
